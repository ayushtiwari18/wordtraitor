# WordTraitor Music Files

## Required Music Files

Place your music files here with the following names:

### 1. `lobby.mp3`
- **Phase:** Lobby/Waiting
- **Mood:** Calm, relaxed, casual
- **Tempo:** Slow
- **Style:** Lo-fi, ambient, chill
- **Suggested:** Light background music that doesn't distract

### 2. `gameplay.mp3`
- **Phases:** Whisper, Hint Drop, Debate, Voting
- **Mood:** Tense, mysterious, focused
- **Tempo:** Medium to fast
- **Style:** Suspense, thriller, puzzle
- **Suggested:** Building tension, keeps players engaged

### 3. `results.mp3`
- **Phase:** Reveal, Results
- **Mood:** Dramatic, climactic
- **Tempo:** Varies
- **Style:** Victory/defeat themes
- **Suggested:** Short, impactful track for final reveal

## Where to Find Free Music

### Recommended Sources (Royalty-Free):

1. **Incompetech (Kevin MacLeod)**
   - https://incompetech.com/music/
   - License: CC BY 4.0 (Free with attribution)
   - Great variety, easy to use

2. **Purple Planet**
   - https://www.purple-planet.com/
   - Free for personal/commercial use
   - Game music specifically

3. **Bensound**
   - https://www.bensound.com/
   - Free with attribution
   - High quality tracks

4. **FreePD (Public Domain)**
   - https://freepd.com/
   - 100% free, no attribution required

5. **YouTube Audio Library**
   - https://studio.youtube.com/
   - Many free tracks with filters

## File Specifications

- **Format:** MP3
- **Bitrate:** 128-192 kbps (balance between quality and file size)
- **Duration:** 
  - Lobby: 2-3 minutes (loops)
  - Gameplay: 3-5 minutes (loops)
  - Results: 30-60 seconds (doesn't loop)
- **Target Size:** 500KB - 2MB per file

## Quick Start (Placeholders)

If you don't have music files yet, the game will work fine - it just won't play any music. The AudioManager handles missing files gracefully.
